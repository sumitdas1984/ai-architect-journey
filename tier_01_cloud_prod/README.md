# Tier 1: Cloud Infrastructure (AWS)

Hands-on learning of core AWS services through project-based work.

## Goal

Build practical expertise with these AWS services:

| Service | Service | Service |
|---------|---------|---------|
| S3 | IAM | Lambda |
| RDS | API Gateway | Step Functions |
| Bedrock | SageMaker | ECS + ECR |

## Approach

Learn by building. Projects are added as understanding grows — no fixed structure or roadmap upfront. Each project is self-contained and reflects where things are at that point in the learning journey.
