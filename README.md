# AI Architect Journey

A structured learning path from cloud infrastructure to advanced agentic AI systems.

## Roadmap

### Tier 1: Cloud & Production (tier_01_cloud_prod)
AWS Lambda, API Gateway, S3, Bedrock, CI/CD deployment patterns.

### Tier 2: Core AI (tier_02_core_ai)
Prompt Engineering, RAG, Embeddings, Vector Databases.

### Tier 3: Agentic AI (tier_03_agentic)
LangGraph, Agents, MCP, Orchestration frameworks.

### Tier 4: Ops & Observability (tier_04_ops)
Evaluation frameworks, Langfuse, Observability.

### Tier 5: Architecture (tier_05_architecture)
System design documents and architectural patterns.
